    Guessing Game


    This is a guessing game that automatically generates a random number once loaded and allows the player to take 10 guesses to find the randomly generated number.


    Application specification:


    • Using CSS to make the webpage responsive.
    • Using JavaScript to generate the random number.
    • JavaScript also allows the user to enter 10 guesses and will compare each guess to the randomly generated number between 1 to 100.
    • If the guessed number is lower that the randomly generated number, the user will get a message at the bottom that the guessed number is too low.
    • If the guessed number is higher than the randomly generated number, the user will get a message at the bottom saying that the guessed number is too high.
    • If the user guesses correctly, they will get a message at the bottom confirming that the number is correct and displaying it.
