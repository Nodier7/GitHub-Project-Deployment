    Number Guessing Game


    This is a guessing game application that automatically generates a random number once loaded and allows the player to take 10 guesses to find the randomly generated number.


    Application specification:


    • Using CSS to make the webpage responsive.
    • Using JavaScript to generate the random number.
    • JavaScript also allows the user to enter 10 guesses and will compare each guess to the randomly generated number between 1 to 100.
    • If the guessed number is lower that the randomly generated number, the user will get a message at the bottom that the guessed number is too low.
    • If the guessed number is higher than the randomly generated number, the user will get a message at the bottom saying that the guessed number is too high.
    • If the user guesses correctly, they will get a message at the bottom confirming that the number is correct and displaying it.



    Mybooks Database


    I have created a database of all my books with 3 tables Books, Authors and Subjects. I have used Python and embedded SQlite3 queries to perform the CRUD operations (Create, Read, Update and Delete). Using Repl.it, the database is accessed by the menu that gives 7 options to the user. The 3 tables have been created in advance and the DBMS does not allow the user to crteate a new table.


    CRUD Operations


    1. This option is the Read operation that allows the user to display each table by entering the table name.
    2. The second option is part of the Create operation that allows the user to add records to the Authors table.
    3. The third one is also part of the Create operation that allows the user to add records to the Books table.
    4. The fourth one is also part of the Create operation that allows the user to add records to the Subjects table.
    5. The fifth option is part of the Update operation that allows the user to update the records of each table.
    6. The sixth option is part of the Delete operation that allows the user to delete a record of each table.
    7. The seventh option is self-explanatory, it allows the user to exit the application.
